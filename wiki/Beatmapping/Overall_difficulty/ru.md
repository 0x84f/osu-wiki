# Overall difficulty

*Подробнее о рекомендованных значениях OD: [Ranking criteria](/wiki/Ranking_Criteria)*

**Overall difficulty** (***OD***, рус. *общая сложность*) —  это одна из настроек сложности, которая определяет, насколько сложно получить высокую [точность](/wiki/Accuracy) на [карте](/wiki/Beatmap).  Этот параметр варьируется от 0 до 10; чем выше его значение, тем точнее нужно нажимать. С тех пор как точность нажатий оказывает воздействие на полосу здоровья, общая сложность тоже будет влиять на сложность прохождения карты.

## Тайминг

При более высоких значениях OD, создаются более короткие временные промежутки для прохождения [игровых объектов] ](/wiki/Hit_Objects), как в целом, так и для получения высокой оценки. Ширина окна, определяющая точность попадания, высчитывается по следующей формуле:

| Счёт | Окно попадания |
| --: | :-- |
| 50 | `400ms - 20ms * OD` |
| 100 | `280ms - 16ms * OD` |
| 300 | `160ms - 12ms * OD` |

Игроку будет засчитан промах, если он не попадёт в окно попадания 50.

## Слайдеры и спиннеры

В [osu!standard](/wiki/Game_mode/osu!) слайдерам будет присуждаться оценка 300 до тех пор, пока игрок не выходит за пределы окна попадания 50. Эта особенность иногда называется *slideracc*, а также удалена в ScoreV2<!-- TODO: link -->.

Общая сложность также влияет на спиннеры: их надо крутить быстрее, чтобы успеть вовремя пройти. В [osu!taiko](/wiki/Game_mode/osu!taiko) схожая ситуация: игроку нужно будет нажимать быстрее, чтобы пройти денден. Количество вращений в секунду, необходимое для прохождения спиннера, определяется по следующей формуле:<!-- TODO: this is probably totally wrong with recent spinner changes -->

- OD < 5: `5 - 2 * (5 - OD) / 5`
- OD = 5: `5`
- OD > 5: `5 + 2.5 * (OD - 5) / 5`

## Нотлок

В osu!standart нельзя будет пройти игровой объект до тех пор, пока прошлый не будет пройден (или упущен после пропуска окна попадания). Это приведёт к **нотлоку** — блокировке ноты и последующему промаху, даже если игрок попадёт по ней идеально в ритм. Наиболее часто это происходит на картах с низкими значениями OD, потому что окна попаданий игровых объектов чаще перекрываются. 

## Игровые модификаторы

Есть четыре мода, которые изменяют общую сложность при активации:

- [Easy](/wiki/Game_modifier/Easy): уменьшает значение OD вдвое.
- [Hard Rock](/wiki/Game_modifier/Hard_Rock): умножает значение OD на 1.4 (максимальное значение — 10). 
- [Double Time](/wiki/Game_modifier/Double_Time): не влияет на OD, но из-за 50% ускорения игры, окна попадания становятся на 33% короче.
- [Half Time](/wiki/Game_modifier/Half_Time): не влияет на OD, но из-за 25% замедления игры, окна попадания становятся на 33% шире.

Хотя Half Time and Double Time не изменяют значение OD, разница в скорости приводит к изменению окон попаданий. Поскольку вычисление каждой оценки отличается, из-за DT окна для 100 и 50 становятся короче, чем обычно, по сравнению с 300; а из-за HT они становятся шире.

## osu!catch

Общая сложность видна при просмотре информации о карте в режиме osu!catch, но она не влияет на игровой процесс.
