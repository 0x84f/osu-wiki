# Горячие клавиши

В osu! есть множество горячих клавиш, которые можно использовать как в игре, так и в режиме ожидания на любом игровом экране. Все они перечислены в списке следующим далее.

**Заметка:** некоторые из комбинаций можно поменять в [настройках](/wiki/Client/Options)! Используемые в данном списке сочетания клавиш установлены по умолчанию.

## Общие

Эти сочетания работают везде:

| Горячая клавиша | Действие |
| :-- | :-- |
| `Alt` + `F4` | Закрыть игру (без предупреждений). Если вы находитесь в режиме игры, редакторе или osu!direct, при использовании этих горячих клавиш вы вернетесь к окну выбора карт или в главное меню игры (при использовании osu!direct) |
| `Alt` + `Стрелки вверх/вниз ` | Изменить общую громкость игры. Используйте клавиши влево или вправо, чтобы изменить громкость эффектов и музыки. |
| `Ctrl` + `Alt` + `Shift` + `S` | Перезагрузить текущий скин. Если выполнить данную команду во время игры или на экране результата, выполнение отложится до того моменты, как Вы вернётесь на экран выбора песни. |
| `Ctrl` + `F11` | Спрятать/показать диаграмму рендеринга кадров. Нажатие на `Ctrl` приостановит её. |
| `F7` | Изменить ограничение по количеству кадров. |
| `F8` | Спрятать/показать чат. |
| `F9` | Спрятать/показать чат и пользователей/друзей онлайн. |
| `F10` | Включить/выключить кнопки мыши (кроме колёсика мыши, оно работает всегда). |
| `F12` | Сделать скриншот. После этого вы можете нажать на уведомление, чтобы открыть папку со скриншотами. |
| `Shift` + `F12` | Сделать скриншот и загрузить его на сервер. Это действие также сохранит скриншот на вашем устройстве. После загрузки скриншот откроется в вашем браузере. |
| `Insert` | [Кнопка-антибосс](https://en.wikipedia.org/wiki/Boss_key). Свернуть игру в системный трей. Это действие также отключит звук от игры. |
| `Shift` + `F1` | Открывает [osu!status](https://twitter.com/osustatus) в вашем браузере. (когда появляется ошибка подключения) |
| `Ctrl` + `O` | Открывает настройки игры. Вы не сможете зайти в настройки универсального оффсета, если вы не находитесь в главном меню. |

## Главное меню

| Горячая клавиша | Действие |
| :-- | :-- |
| `Alt` + `Enter` | Включить/выключить полноэкранный режим. |
| `Стрелки вверх/вниз` | Изменить общую громкость игры. Используйте клавиши влево или вправо, чтобы изменить громкость эффектов и музыки. |
| `Esc` | Вернуться на шаг назад. |
| `P`, `E`, или `O` | Открыть меню игры. |
| `P` три раза | Войти в режим одиночной игры. |
| `P` дважды, затем `M` | Войти в [Мультиплеер](/wiki/Client/Interface/Multiplayer). |
| `E` дважды | Открыть редактор. |
| `O` | Открыть настройки. |
| `Q` | Отобразить диалог выхода из игры. |
| `Esc` | Отобразить диалог выхода из игры, когда все меню закрыты. |
| `F` | Включить счётчик кадров. |

### Проигрыватель

| Горячая клавиша | Действие |
| :-- | :-- |
| `Z` или `F1` | Воспроизвести предыдущую песню. |
| `X` или `F2` | Воспроизвести песню. Если музыка уже играет, начать ее воспроизведение с начала. |
| `C` | Поставить на паузу/воспроизвести. |
| `F4` | Остановить воспроизведение и вернуться в начало. |
| `V` или `F5` | Воспроизвести следующую песню. |
| `J` или `F6` | Открыть список песен. |
| `R` | Воспроизвести случайную песню. |

| `Стрелки вверх/вниз` | Выбрать следующую/предыдущую сложность соответственно. |

| `Shift` + `Стрелки влево/вправо` | Переключиться на следующую/предыдущую группу. |

## Режим игры

| Горячая клавиша | Действие |
| :-- | :-- |
| `+` или `-` | Поменять локальный оффсет (зажав `Alt`, можно увеличить точность). |
| `Esc` | Пауза или выход в меню выбора песен, если вы смотрите чью-то игру или стоит мод Auto или Cinema. |
| `Space` | Пропустить до геймплея. |
| `Tab` | Показать/скрыть таблицу рекордов. |
| `Shift` + `Tab` | Показать/скрыть интерфейс. |
| `H` | Показать/скрыть таблицу рекордов и интерфейс повтора (когда вы смотрите чей-либо повтор или наблюдаете за игрой). |
| `F3` или `F4` | Настроить скорость в osu!mania (работает только в самом начале карты). |
| `Ctrl` + `R` или `` ` `` | Нужно зажать ненадолго. Быстрый перезапуск карты (автоматически пропускает вступление в начале). |

## Режим просмотра повтора

*Примечание: Вы должны использовать версию игры cutting edge, чтобы использовать эти клавиши.*

| Горячая клавиша | Действие |
| :-- | :-- |
| `5` | Поставить повтор на паузу. |
| `6` или `0` | Воспроизвести или поставить скорость на 1x. |
| `7` | Поставить скорость на 2x. |
| `8` | Поставить скорость на 3x. |
| `9` | Поставить скорость на 4x. |

## Режим редактирования карт

*Главная страница: [Редактор карт](/wiki/Client/Beatmap_editor)*

Эти сочетания работают на любом экране редактора.

| Горячая клавиша | Действие |
| :-- | :-- |
| `Z` | Перейти к первой ноте или старту песни. |

| `C` или `Пробел` | Пауза/воспроизведение. |

| `Стрелка вверх` или `Стрелка вниз` | Переключаться между временными отметками. |
| `Shift` + `Стрелка влево` или `Shift` + `Стрелка вправо` | Двигаться вперёд-назад по песне с 4х-кратной скоростью. |
| `Ctrl` + `B` | Добавить закладку на текущей позиции. |
| `Ctrl` + `Shift` + `B` | Удалить закладку с текущей позиции. |
| `Ctrl` + `Стрелки влево/вправо` | Перейти к предыдущей/следующей закладке. |
| `Ctrl` + `L` | Перезагрузить до последней сохраненной версии (частично). Перезагружает файлы `.osu` и `.osb`. |
| `Ctrl` + `Shift` + `L` | Перезагрузить до последней сохраненной версии (полностью). Перезагружает все файлы карты |
| `Ctrl` + `O` | Открыть окно выбора сложности. |
| `Ctrl` + `S` | Сохранить текущее состояние карты. |
| `Ctrl` + `N` | Удалить все объекты на карте. |
| `Ctrl` + `Стрелки вверх/вниз` | Увеличить/уменьшить скорость воспроизведения на 25%. |
| `Ctrl` + `Shift` + `Стрелки вверх/вниз` | Увеличить/уменьшить скорость воспроизведения на 5%. |
| `F1` | Переключиться в режим [compose](/wiki/Client/Beatmap_editor/Compose). |
| `F2` | Переключиться в режим [design](/wiki/Client/Beatmap_editor/Design). |
| `F3` | Переключиться в режим [timing](/wiki/Client/Beatmap_editor/Timing). |
| `F4` | Открыть [настройки песни](/wiki/Client/Beatmap_editor/Song_setup). |
| `F5` | Открыть режим тестирования. |
| `F6` | Открыть панель настроки тайминга. |
| `Ctrl` + `P` | Добавить ненаследуемую тайминг секцию. Если вы находитесь не в режиме timing, это действие также откроет диалоговое окно настроек тайминга. |
| `Ctrl` + `Shift` + `P` | Добавить наследуемую тайминг секцию. Это действие также откроет диалоговое окно настроек тайминга. |
| `Ctrl` + `I` | Удалить текущую тайминг секцию. Не работает для самой первой ненаследуемой секции. |
