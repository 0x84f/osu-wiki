# Canal de mise à jour

osu! dispose de plusieurs **canaux de mise à jour**. Ces canaux de mise à jour sont les différentes versions du jeu, qui reçoivent des mises à jour à des rythmes différents. Afin de détecter rapidement les problèmes potentiels, les nouvelles fonctionnalités et les modifications sont intégrées dans les versions de test avant d'être transmises au client stable.

| Canal de mise à jour | Notes |
| :-- | :-- |
| Stable | Version actuelle du client ; [ne reçoit plus de nouvelles fonctionnalités](https://fr.wikipedia.org/wiki/Feature_freeze) ; reçoit les mises à jour après la version Beta. |
| Beta | Reçoit les mises à jour après la version Cutting Edge ; ne reçoit plus de nouvelles fonctionnalités. |
| Cutting Edge | Reçoit les mises à jour les plus récentes ; ne reçoit plus de nouvelles fonctionnalités. |
| [Lazer](Lazer) | Client du jeu réécrit ; avec le développement le plus actif et [beaucoup de différences et de nouvelles fonctionnalités](/wiki/Client/Release_stream/Lazer/Feature_comparison). |
