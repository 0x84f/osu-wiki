# Ранк / ранг

Иностранный вариант слова «Rank», в русском языке имеет несколько значений и написаний:

**Ранг** может означать:

- [Оценка](/wiki/Gameplay/Grade) — результат игрока на карте.
- [Рейтинг игрока](/wiki/Ranking) — позиция в онлайн-таблице рекордов.

**Ранк** означает процесс получения таблицы рекордов для своей карты (прохождение [процедуры ранкинга](/wiki/Beatmap_ranking_procedure)).
