# Random (мод)

![Значок мода RD](/wiki/shared/mods/RD.png "Значок мода Random (RD)")

*Полный список модов: см. [Игровой модификатор](/wiki/Gameplay/Game_modifier)*

## Краткая информация

- Сокращение: RD
- Тип: особый мод
- Множитель очков: 1.00x
- Горячая клавиша по умолчанию: `X`
- Описание: `Ноты перемешиваются.`
- Режимы игры: ![][osu!mania]

## Описание

**Random** — [игровой модификатор](/wiki/Gameplay/Game_modifier), для режима [osu!mania](/wiki/Game_mode/osu!mania). Мод перемешивает ноты при каждой игре на карте, обеспечивая интересный опыт, особенно для прошедших определённые карты множество раз. Random не изменяет длину нот.
