# Оценка

**Оценки** набор букв, обозначающий результат игрока на определённой [карте](/wiki/Beatmap). Они показаны вместе с [очками](/wiki/Gameplay/Score) в таблице результатов.

От наименьшей к наибольшей, существуют оценки D, C, B, A, S, и SS. Серебрянные S и SS можно получить только с использованием модификаторов [Hidden](/wiki/Game_modifier/Hidden), [Flashlight](/wiki/Game_modifier/Flashlight), или [Fade In](/wiki/Game_modifier/Fade_In).

## osu!

| Оценка | Условие |
| :-: | :-- |
| SS | 100% точность |
| S | Более 90% 300, не более 1% 50, и нет промахов |
| A | Более 80% 300 без промахов **ИЛИ** более 90% 300 |
| B | Более 70% 300 без промахов **ИЛИ** более 80% 300 |
| C | Более 60% 300 |
| D | В остальных случаях |

## osu!taiko

| Оценка | Условие |
| :-: | :-- |
| SS | 100% точность |
| S | Более 90% GREAT и без промахов |
| A | Более 80% GREAT и без промахов **ИЛИ** более 90% GREAT |
| B | Более 70% GREAT и без промахов **ИЛИ** более 80% GREAT |
| C | Более 60% GREAT |
| D | Любой другой результат |

## osu!catch

| Оценка | Условие |
| :-: | :-- |
| SS | 100% точность |
| S | От 98.01% до 99.99% точности (S можно получить даже с несколькими промахами, как и в osu!mania) |
| A | От 94.01% до 98.00% точности |
| B | От 90.01% до 94.00% точности |
| C | От 85.01% до 90.00% точности |
| D | Любая точность ниже 85% |

| C | Более 70% точности |
